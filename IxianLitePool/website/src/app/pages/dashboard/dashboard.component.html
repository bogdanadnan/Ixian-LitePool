Getting Started

You'll need an Ixian Wallet Address and Ixian mining software.
The fastest way to get the wallet address is by using Spixi or Ixian LiteWallet.
Remember to backup your wallet!
You can choose most suitable mining software for you below.

Pool Address:

IxiMiner - mixed CPU/GPU Miner

Latest release of IxiMiner and instructions can be found on IxiMiner GitHub page.

We highly recommend reading additional instructions from here Iximiner for Beginners, it will help you get started and answer the most common questions related to iximiner.


IxiWatt Easy Miner for Windows - GUI frontend for IxiMiner

Latest release of IxiWatt can be found on Project Ixian GitHub page.


Official CPU only miner

Latest release of Official CPU miner can be found on Project Ixian GitHub page.

IxianMiner command

IxianMiner.exe --pool https://ixian-us.kiramine.com --wallet YOUR_WALLET --worker YOUR_WORKERNAME

Make sure you replace YOUR_WALLET with your Ixian wallet address and YOUR_WORKERNAME with a name for your mining rig.

Failover Configuration

To add a second Ixian Pool as Failover, simply add --pool2 "FAILOVER POOL ADDRESS" to your IxianMiner command. For example: --pool2 https://ixian.kiramine.com
